my game: Cage Flight!

First rule of Flight Club: tell everyone about Flight Club!

This weekend we made a space fighter game for the Ludum Dare competition.

There are over 1163 other entries as I type this, and a few more might turn up before the deadline.

Here’s the link to the game: http://www.ludumdare.com/compo/ludum-dare-24/?action=preview&uid=10313

And here’s a gameplay video:

The Ludum Dare theme was evolution.  Now we figured that everyone will be doing evolving kind of things. Soups and stuff.  Oh, and platform games as ever.

So we set out to think about unnatural selection; this became a discussion about dueling and then cage fights and then the vision emerged: multi-player on-line space fighter dueling in a cage!

The game uses webGL, the new Web Audio API and websockets.  Its written in Javascript without libs.  (The server is Python.)  Any jitter is because of lag between you and my server in Sweden, sorry.

What are you waiting for?  Tell your friends to meet you in the cage right now!!!
